# AI-Based Hospital Management System

## 📌 Description
This project is a web-based application developed using Streamlit to demonstrate an AI-Based Hospital Management System. It provides role-based access for Patients and Doctors to manage hospital activities efficiently.

## 🚀 Features

### 👤 Patient
- Register / Login
- Book Appointment with specialists:
  - General Physician
  - Dermatologist
  - Cardiologist
  - Orthopedic
  - Pediatrician
  - ENT Specialist
- View Medical Reports
- View Appointments
- Chatbot for basic queries

### 👨‍⚕️ Doctor
- View Upcoming Appointments
- Access Patient Details
- View Past Reports
- Write Prescription
- Set Availability

## 🛠️ Technology Used
- Python
- Streamlit

## 🌐 Deployment
This project can be deployed using Streamlit Cloud.
